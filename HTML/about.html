ABOUT NOVACART

A polished front-end ecommerce experience built for real-world practice.

NovaCart demonstrates product APLs, persistent carts, pagination, account flows, and responsive Bootstrap design in a simple static project.

Fast Catalog

Products are loaded from JSON and rendered with efficient search, filter, sort, and pagination controls.

Persistent Cart

Cart items and quantities are saved in browser storage, so the shopping flow survives refreshes.

Account Flow

Registration and login include validation, duplicate checks, password hashing, and session handling.

Built with practical standards

The app keeps concerns separated across markup, styling, catalog logic, and authentication logic. It is easy to host as a static site and easy to connect to a real backend later.
