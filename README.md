# Nova_Cart

A modern e-commerce shopping cart application with a sleek interface for browsing and purchasing products.

## 📋 Features

- **Product Browsing**: Browse products across multiple categories
- **Category Filter**: Filter products by Beauty, Electronics, Fashion, Home, Outdoor, and Sports
- **Search Functionality**: Search for specific products
- **Responsive Design**: Mobile-friendly layout
- **Shopping Cart**: Add/remove items from cart
- **Bootstrap Icons**: Modern icon library integration

## 📁 Project Structure

```
Nova_Cart/
├── HTML/
│   └── nova.html          # Main HTML file
├── CSS/
│   └── nova.css           # Styling
├── JS/
│   └── nova.js            # JavaScript functionality
├── nova_imgs/
│   └── background.png     # Background image
└── README.md
```

## 🛠️ Technologies Used

- **HTML5**: Page structure
- **CSS3**: Styling and responsive design
- **JavaScript**: Interactive features
- **Bootstrap Icons**: Icon library
- **Font Awesome**: Additional icons

## 🚀 Installation

1. Clone the repository:
```bash
git clone https://github.com/nayankumarnk1010/Nova_Cart.git
```

2. Navigate to the project directory:
```bash
cd Nova_Cart
```

3. Open `HTML/nova.html` in your web browser

## 💡 Features Overview

### Navigation
- Logo/branding

- Dropdown menu with multiple categories

- Clean and modern design

## 📝 Usage

1. Open the application in a web browser
2. Use the search bar to find products
3. Select categories from the dropdown menu
4. Add items to your shopping cart
5. Proceed to checkout
