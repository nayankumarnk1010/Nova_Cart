# City General Hospital - Hospital Management System Website

A complete, responsive website for a hospital management system featuring modern design and comprehensive healthcare information.

## Features

### 🏥 **Complete Hospital Website**
- Professional medical-themed design
- Fully responsive layout (mobile & desktop)
- Modern CSS with medical color scheme
- Interactive JavaScript elements

### 🎨 **Design Elements**
- **Color Scheme**: Medical blues (#0066cc, #0099ff), accent green (#00ff88), clean whites
- **Typography**: Clean, readable fonts
- **Images**: High-quality medical and hospital photography
- **Layout**: Grid-based responsive design

### 📱 **Sections Included**
1. **Header**: Fixed navigation with smooth scrolling
2. **Hero**: Compelling introduction with call-to-action buttons
3. **Services**: 6 key medical services (Emergency, Cardiology, Neurology, Dentistry, Pediatrics, Laboratory)
4. **About**: Hospital history and commitment to excellence
5. **Doctors**: Profiles of 4 expert medical professionals
6. **Contact**: Contact information and message form
7. **Footer**: Comprehensive site links and information

### ⚡ **Interactive Features**
- Smooth scrolling navigation
- Hover effects on cards and buttons
- Contact form with validation
- Scroll-based header transparency
- Mobile-responsive menu

## Usage
1. Open `index.html` in any modern web browser
2. Navigate using the fixed header menu
3. Use the contact form to send messages
4. All links are functional for smooth scrolling

## Customization
- Modify colors in the CSS section of `index.html`
- Replace placeholder images with your own
- Update hospital information and services
- Add additional pages or sections as needed
